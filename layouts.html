Grid System

The grid is the building blocks used to provide layout service on pages.

It uses a series of classes for .containers, .row and .col to layout and align content. It’s built with flexbox and is fully responsive.

The grid can accomodate up to 12 columns.

Embed Videos

For better watching experience, always consider the aspect ratio of the video so that it won't affect the quality.

In embeding video, use the following HTML tags <iframe>, <embed> and <video>.

Tip

Since the design library is built on top of Boostrap, you don’t need frameborder="0" on your iframe's. 

																									
<div class="ratio ratio-16x9">
  <iframe class="embed-responsive-item" src="https://www.youtube.com/embed/VyhLRJVoIrI"></iframe>
 </div>

Breakpoints

Setting up the web application dimensions in different screen sizes of the device. These breakpoints will be used to create a flexible interfaces regardless the site is viewed in mobile, tablets or desktop.

Core Concepts

There are six default breakpoints (grid tiers) to make a responsive design. The following are the identified screen size dimensions.

Default Breakpoints
Breakpoint Class infix Dimensions
X-Small None <576px
Small sm ≥576px
Medium md ≥768px
Large lg ≥992px
Extra large xl ≥1200px
Extra extra large xxl ≥1400px
Note: Breakpoint width are multiples of 12 and customizable via Sass— in a Sass map in the _variables.scss stylesheet.

Breakpoints are supported through using a handful of media queries that will make the page responsive as viewport changes.

Viewport Breakpoint Notes
Min-width
  • @include media-breakpoint-up(sm) { ... }
  • @include media-breakpoint-up(md) { ... }
  • @include media-breakpoint-up(lg) { ... }
  • @include media-breakpoint-up(xl) { ... }
  • @include media-breakpoint-up(xxl) { ... }
 For xs breakpoint, @media (min-width: 0) { ... } is used.
Max-width
  • @include media-breakpoint-down(sm) { ... }
  • @include media-breakpoint-down(md) { ... }
  • @include media-breakpoint-down(lg) { ... }
  • @include media-breakpoint-down(xl) { ... }
  • @include media-breakpoint-down(xxl) { ... }
 For xs breakpoint, @media (max-width: 0) { ... } is used.
